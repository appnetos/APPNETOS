Если вы предоставляете области для пользователей, зарегистрированных в вашем веб-приложении, вы можете использовать эту форму, чтобы позволить новым пользователям зарегистрироваться.
Приложение использует систему APPNET OS для создания новых пользователей. Это дает вам два способа регистрации новых пользователей. С проверкой электронной почты и простой логин без подтверждения адреса электронной почты.
AppNET OS Mailer используется для подтверждения адреса электронной почты. Вновь зарегистрированный пользователь получит электронное письмо со ссылкой для подтверждения при регистрации. В настройках APPNET OS Mailer должен быть настроен почтовый ящик для приписки к регистрационной форме администратора APP. Почтовый ящик должен быть установлен над приложением. По соображениям безопасности APPNET OS Mailer никогда не должен устанавливаться статично.

- Простой способ предоставить пользователям регистрацию в вашем веб-приложении
- Легкая конфигурация
- Предлагает два варианта индивидуальной регистрации. С подтверждением электронной почты и без нее
- Многоязычный
- Область применения на 6 языках. Английский, немецкий, испанский, французский, итальянский, русский
- Область администратора на 6 языках. Английский, немецкий, испанский, французский, итальянский, русский
- Контейнерное приложение. Размер настраивается через настройки приложения
- Бесплатно для коммерческих и некоммерческих проектов

Использовать:
- При регистрации с подтверждением электронной почты, настроить почтовый ящик для APPNET OS Mailer, назначить почтовый ящик в настройках администратора приложения приложения, место почтового ящика над этим приложением.
- Поместите приложение в подходящее место

Зависимости:
- APPNET OS Mailer, для регистрации с подтверждением электронной почты
