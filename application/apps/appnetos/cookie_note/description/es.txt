Cookie Note le ofrece la posibilidad de proporcionar a sus usuarios la configuración para el uso de diferentes tipos de cookies.
La aplicación utiliza el sistema de cookies integrado de APPNET OS. Sólo se permiten las cookies que el usuario acepta. Esto solo funciona para aplicaciones que utilizan el sistema de cookies integrado.
Se informa al usuario cuando el sistema intenta establecer una cookie no autorizada.
En la Configuración del administrador de aplicaciones puede especificar qué tipos de cookies bloquear.
