Cookie Note дает возможность предоставлять пользователям настройки для использования различных типов файлов cookie.
Приложение использует встроенную систему cookie APPNET OS. Разрешены только файлы cookie, на которые соглашается пользователь. Это работает только для приложений, которые используют встроенную систему cookie.
Пользователь информируется, когда система пытается установить несанкционированное файл cookie.
В настройках администратора приложения можно указать, какие типы файлов cookie блокировать.

- Автоматически генерируемая нота cookie
- Легкая установка
- Легкая конфигурация
- Контейнерное приложение. Размер приложения может быть легко настроен в настройках приложения
- Регулировка цвета через классы CSS в настройках приложения
- Использует систему блокировщика cookie APPNET OS
- Информирует пользователя, когда система пытается установить несанкционированное cookie
- Многоязычный
- Область администратора на 6 языках. Английский, немецкий, испанский, французский, итальянский, русский
- Область применения на 6 языках. Английский, немецкий, испанский, французский, итальянский, русский
- Бесплатно для коммерческих и некоммерческих проектов

Использовать:
- Добавить в качестве статического приложения в верхней или нижней
- Настройка настроек отображения в настройках администратора приложения
