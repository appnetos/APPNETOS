С заголовком APPNET OS вы получаете возможность легко создать большую область заголовка для вашего веб-приложения.
Вы можете установить логотип и анимированные значки для социальных медиа. Приложение многоязычное. Значки логотипа и ссылки могут быть определены отдельно для каждого языка. Цвета приложения можно легко изменить с помощью настроек администратора приложения. Приложение использует цвета, указанные Bootstrap, так что внешний вид всегда соответствует вашему текущему дизайну.

- Быстрый и простой в настройке области головы
- Анимированные иконки для социальных медиа
- Цвета полностью настраиваются через область администратора
- Контейнерное приложение. Размер настраивается через настройки приложения
- Многоязычный, для каждого языка вы можете определить свой собственный логотип и отдельные значки для социальных медиа
- Область администратора на 6 языках. Английский, немецкий, испанский, французский, итальянский, русский
- Возможна многократная установка и конфигурация
- Бесплатно для коммерческих и некоммерческих проектов

Использовать:
- Место в подходящем месте
- Настройка логотипа, значков для социальных сетей и цветов с помощью настроек администратора приложений
