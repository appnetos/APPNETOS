Сделайте ваше веб-приложение привлекательным.
С помощью приложения для смены фона можно легко создавать меняющиеся фоны.
Фон может иметь простой цвет или фоновое изображение. Для фонового изображения можно настроить повторение и размер. Фон ы может быть назначен либо по отдельности, либо поочередно жидкостным контейнерам, контейнерам или приложениям. Приложение генерирует необходимые классы CSS для этого. Они должны быть добавлены в приложения в настройках приложения. Используйте это приложение в качестве статического приложения или добавьте его на страницу с помощью CMS Manager.

- Легкая установка
- Легкая конфигурация с помощью настроек администратора приложения
- Фоны могут быть добавлены в качестве одного фона или в качестве повторяющихся фон
- Область администратора на 6 языках. Английский, Немецкий, Испанский, Французский, Итальянский, Русский
- Возможна многократная установка и конфигурация
- Бесплатно для коммерческих и некоммерческих проектов

Использовать:
Либо добавить в статические приложения или добавить в качестве приложения на странице вы используете.
Настройка фона в настройках администратора приложений.
В настройках приложения приложения используют фоны, добавляя сгенерированные классы CSS.
