APPNET OS Конец контейнера является важным системным приложением.
Компонент завершает отзывчивый контейнер и заставляет систему построить новый.
Если это приложение расположено между несколькими контейнерными приложениями, для каждого приложения создается отдельный контейнер.
Это делает приложение важным компонентом системы.
Приложение может быть установлено и использоваться так часто, как хотелось бы.

Приложение является стандартным компонентом системы и присутствует в каждой версии APPNET OS.
Если приложение не доступно или потеряно, его можно скачать или обновить здесь бесплатно в любое время.
