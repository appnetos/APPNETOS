Создайте многоязычное меню для веб-приложения в кратчайшие сроки.
Меню использует Bootstrap Navbar и может быть полностью настроен из области администрирования приложений. Введите только текст для соответствующего языка и ссылки. Ссылки могут использовать идентификаторы CMS, поэтому они всегда правильно высващаемы на любом языке.
В настройках администратора приложения каждой части меню может быть присвоен цвет загрузки, поэтому он всегда соответствует вашей текущей теме.
Кроме того, в меню есть кнопка входа и интегрированная форма входа, которая позволяет пользователям войти в ваше веб-приложение.
Приложение представляет собой контейнерное приложение. Просто установите желаемый размер через настройки приложения. App Кэш сохраняет результат вывода в базе данных, что делает рендеринг и выход очень быстро.

- Быстрый и простой в настройке меню бар
- Кнопка регистрации со встроенной регистрационной формой
- Многоязычное меню может быть создано отдельно для каждого языка
- Цвета полностью настраиваются через область администратора
- Контейнерное приложение. Размер настраивается через настройки приложения
- Область администратора на 6 языках. Английский, немецкий, испанский, французский, итальянский, русский
- Возможна многократная установка и конфигурация
- Бесплатно для коммерческих и некоммерческих проектов

Использовать:
- Настройка меню с помощью настроек администратора приложения
- При настройке меню убедитесь, что вы отключите настройки кэша приложения, в противном случае результат, хранящийся в кэше, будет выходным
- Поместите приложение в подходящее место
