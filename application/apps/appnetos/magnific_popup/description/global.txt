С помощью этого приложения вы добавляете плагин J'sa "Magnific всплывающее окно в простейшем способе.
Magnific Popup - это лайтбокс j's'ry со многими большими возможностями.

Дополнительную информацию о функции можно найти на главной странице.
https://dimsemenov.com/plugins/magnific-popup/

Благодаря загрузке и установке, у вас сразу же есть много новых инструментов, доступных для вашего HTML-контента.

- Одноизображение Лайтбокс
- Лайтбокс Галерея
- Увеличить Галерея
- Всплывающее окно с видео или картами
- Диалог с анимацией CSS
- Всплывающее окно с формой
- всплывающее окно AJAX
- Модальный всплывающее окно

Использовать:
Поместите приложение в верхней части страницы.
Если вы используете кэш CSS и JavaScript, размещение не требуется. Приложение только должно быть активировано в этом случае.
