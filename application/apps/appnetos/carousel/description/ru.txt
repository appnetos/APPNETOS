Используйте приложение и просто создать Карусель Bootstrap фотографии.
Просто управляйте фотографиями карусели из панели администратора приложений и поместите приложение в любом месте вашего веб-приложения.
Дисплей отзывчив, а размер приложения свободно настраивается через настройки приложения.
Для каждого изображения можно определить ссылку.
Приложение многоязычное. Дополнительный файл может быть использован для каждого изображения, для каждого языка.
Настройки можно использовать для влияния на время отображения и элементы управления.

- Легкая установка
- Легко управлять
- Многоязычный. Для каждого языка можно использовать отдельное изображение.
- Ссылка может быть определена для каждого изображения
- Отображение параметров времени и элементов управления
- Контейнер приложение. Размер настраивается через настройки приложения
- Область администратора на 6 языках. Английский, Немецкий, Испанский, Французский, Итальянский, Русский
- Возможна многократная установка и конфигурация
- Бесплатно для коммерческих и некоммерческих проектов

Использовать:
- Место в подходящем месте
- Настройка изображений, ссылок и настроек через область администратора
