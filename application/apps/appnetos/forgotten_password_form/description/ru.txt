С помощью этого приложения вы даете пользователям возможность сбросить пароль.
При вводе адреса электронной почты пользователю отправляется ссылка для сбросить пароль. Приложение использует APPNET OS Mailer. Это позволяет быстро интегрировать сядки и настройку. В настройках администратора приложения должен быть назначен почтовый ящик почтового ящика APPNET OS. Почтовый ящик должен быть установлен над приложением.

- Быстрая и несложная интеграция
- Многоязычный
- Область администратора на 6 языках. Английский, немецкий, испанский, французский, итальянский, русский
- Область применения на 6 языках. Английский, немецкий, испанский, французский, итальянский, русский
- Контейнерное приложение. Размер приложения может быть легко настроен в настройках приложения
- Использует APPNET OS Mailer
- Бесплатно для коммерческих и некоммерческих проектов

Использовать:
- Установка приложения
- Наконфигурировать почтовый ящик Mailer
- Присвоить почтовый ящик Mailer в настройках администратора приложений
- Установите почтовый ящик над приложением

Зависимости:
